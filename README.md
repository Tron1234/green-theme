## 介绍
该主题是我自定义主题基于hbuilder代码颜色和添加背景护眼色[#cce8cf]编写的，希望大家在敲代码的同时也要爱护自己的眼睛。

### 首先安装vsce
`npm install -g vsce`
### 使用vsce命令打包
`vsce package # package`
### 使用vsce命令发布
`vsce publish [1.0.0] // publish后面可添加版本号`
